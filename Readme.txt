Данная ветка предназначена для программы, которая считывает данные о показаниях счётчиков с сервера oiot.ru и отправляет их на сервер MOS.RU с помощью SMS.

В данной архитектуре смартфон на Android выступает в роли промежуточного сервера. В определённое время он автоматически запускает программу, выполняющую пересылку данных.

Программа написана на Python и запускается в среде Termux.

Схема работы следующая:

Приложение Termux находится в автозапуске Android;

При старте Termux автоматически запускается cron;

В cron настроено расписание, по которому запускается основная Python-программа;

Программа получает показания с oiot.ru, обрабатывает их и пересылает через SMS на MOS.RU.

Дальнейшее развитие планирую по мере увеличения интереса к проекту.
