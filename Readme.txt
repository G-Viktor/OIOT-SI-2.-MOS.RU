Данная ветка создана для программы, которая считывает данные о показаниях счётчиков с сервера oiot.ru и через СМС передаёт показания на сервер MOS.RU.
В данном случае телефон на Android выполняет роль сервера, который в определённое время занимается пересылкой данных.
Программа написана на python. Запускается программа из под терминала Termux.
Примерная схема работы: Termux находится в автозапуске, так жe в след за ним запускается crone, в котором стоит задача на запуск самой программы.
